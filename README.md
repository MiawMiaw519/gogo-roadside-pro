# GoGo Roadside Pro

Site web professionnel de dépannage et d'assistance automobile conçu pour un professionnel du secteur du remorquage et du dépannage routier.

## Accès au site

Le site est disponible à l'adresse suivante : https://miawmiaw519.github.io/gogo-roadside-pro/

Le projet est hébergé via GitHub Pages, permettant une mise en ligne rapide et accessible depuis n'importe quel appareil. :contentReference[oaicite:0]{index=0}

---

## Présentation

GoGo Roadside Pro est une plateforme vitrine destinée à présenter les services d'un professionnel du dépannage automobile.

L'objectif principal du site est de permettre aux clients de :

- Découvrir les services proposés
- Obtenir rapidement les informations de contact
- Demander une assistance en cas de panne
- Trouver une solution de dépannage ou de remorquage
- Accéder facilement au site depuis un ordinateur, une tablette ou un smartphone

---
